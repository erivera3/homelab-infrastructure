# 🖥️ Homelab Infrastructure Evolution
### From Legacy Hardware to Enterprise-Style Network Lab

---

## 🚀 Overview

This project documents the **design, teardown, and iterative rebuild** of a homelab environment driven by real-world constraints:

- power consumption
- heat and airflow
- noise
- reliability
- scalability

The lab evolved from **repurposed legacy desktops** into a **modern, virtualized, enterprise-style environment** with centralized identity, DNS, storage, and firewalling.
